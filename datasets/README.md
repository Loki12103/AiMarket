# Datasets Folder

This folder contains all datasets used in the project.

## Available Datasets

### 1. **Amazon DataSheet - Pradeep.xlsx**
- Source: Amazon product reviews
- Contains: Product Name, Comment, Star Rating, Date of Review, Category, Sentiment
- Used by: `data_cleaning.py`, `merge_datasets.py`

### 2. **flipkart_product.csv**
- Source: Flipkart e-commerce
- Contains: ProductName, Review, Summary, Rate
- Used by: `merge_datasets.py`

### 3. **sentimentdataset.csv**
- Source: Sentiment analysis dataset
- Contains: Text, Sentiment, Timestamp, User, Platform data
- Used by: `categorize_products.py`

## Output Files (Generated)

### 1. **amazon_dataset_cleaned.csv**
- Generated by: `data_cleaning.py`
- Contains cleaned Amazon data

### 2. **sentimentdataset_cleaned.csv**
- Generated by: `data_cleaning.py`
- Contains cleaned sentiment data

### 3. **combined_cleaned_data.csv**
- Generated by: `merge_datasets.py`
- Contains merged Amazon + Flipkart data

### 4. **sentimentdataset_with_categories.csv**
- Generated by: `categorize_products.py`
- Contains sentiment data with assigned categories
